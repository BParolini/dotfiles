# Firefox configuration

## Company computer
Access the URL `about:config` and alter the following configuration:
- `security.enterprise_roots.enabled = true`

## Android version
Access the URL `about:config` and alter the following configuration:
- `general.useragent.override = Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:68.0) Gecko/20100101 Firefox/68.0`
	- The version must be adjusted as the browser is updated.
